# How to use

Copy keys.env file from main directory and add your API-keys.

# How to Serve the App

# Notebook Version
Serve notebook version in terminal with:

panel serve crypto_prices_over_time.ipynb

# Script Version
Serve script version in terminal with:

bokeh serve crypto_prices_over_time.py
